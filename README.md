# Budgit
Its an web site to manage expense with the group or solo without any tension
A person can track his/her own expenses
even track the budget and expenses between roomates
and yeah even track your expense between your freinds while travelling :)


So the whole Project is Based on PERN Stack + TailwindCSS(Yes its an PERN Stack Project)

P- Postgres Sql is used for database where it stores the whole data user adds form frontend whether its adding groups, expenses, or data regarding login and register basically auth, also contains short term code which basicaly works as an otp while changing Password.

E - Express.js used for routing most of the time.

R + TailwindCSS - React.js is used for frontend and to fetch data from backend also to make the page responsive to get instant data for backend along with that it contains TailwindCSS framework for the desing if the webpage.

N - Node.js is used for backend which has the main logic to collect the data that is shared from frontend and send them to database but not only that it constains the logic where to send how to and the main login of splitting the amount the main concept of the project to share the amount between friends, family, roomates etc. the main logic is coded using node.
